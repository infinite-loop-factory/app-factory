# cat-meetup (냥냥모임)

고양이 돌봄/친구찾기/물품나눔 매칭 서비스를 위한 Expo 앱 기본 골격입니다.

## 현재 포함된 골격
- 사람 회원가입 화면 골격
- 동물 카드(복수) 등록 화면 골격
- 게시물 목록/상세/신청 목록 골격
- 작성자용 게시물 작성/신청자 관리 골격
- 신청자용 매칭 수락 목록 골격
- 도메인 타입 + 목업 데이터 (`src/domain`)

## 실행
```bash
pnpm --filter @infinite-loop-factory/cat-meetup start
```

## 우선 구현 추천
1. 회원가입/로그인 API + 토큰 저장
2. 게시물 CRUD + 신청/취소 API
3. 매칭 상태 전이 정책(대기중/매칭중/매칭완료/실패)
4. 전화번호 노출 규칙(매칭 단계별)
5. 리뷰 작성 가능 시점(일정 다음날) 검증
