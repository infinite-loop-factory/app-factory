### 신규 가입자

1. 앱 실행
2. 사람 회원가입 (필수)
    1. 이름
    2. 핸드폰번호(아이디)
    3. 카카오톡아이디
    4. 비밀번호
    5. 이메일
    6. 성별
    7. 생년월일
    8. 사는 지역(구 로 기준 잡음/ 세세한 주소 공개 안함)선택
        1. 서울시 구로구
        2. 서울시 관악구
    9. 본인 설명/자기소개
3. 동물 등록 (나중에 등록 가능)
4. 동물 카드 작성(복수 생성)
    1. 고양이 이름
    2. 고양이 성별
    3. 고양이 나이
    4. 중성화 여부
    5. 고양이 성격 (개냥이/수줍음/사나움)
    6. 고양이 사진
    7. 고양이 설명
5. 매칭 게시물 목록 조회
    1. 지역기반 조회(구로구/관악구/노원구)
6. 게시물 클릭 후 내용 확인
    1. 제목
        1. 고양이 돌봄
        2. 고양이 친구 찾아요
        3. 고양이 물품 나눔
    2. 카테고리(돌봄/친구찾기/물품나눔)
    3. 일시
        1. 만나기로 한 날짜, 시간(24시간 기준)
            1. 2026년 1월 28일 19시 30분
    4. 내용
        1. 자유롭게 작성
    5. 참여 신청 버튼
    6. 참여 신청 완료 후 diable
7. 신청한 게시물 목록
    1. 신청한 게시물이 나열됨
    2. 매칭되면 오른쪽 박스에 매칭
    3. 대기 중이면 대기
    4. 매칭 실패면 실패
8. 매칭 완료 후
    1. 게시물 작성자의 핸드폰 번호 확인 가능
    2. 자신의 번호는 노출되지 않음
9. 핸드폰으로 각자 연락 후 고양이 관리
10. 완료된 날짜 기준으로 다음날부터 리뷰 작성 가능

### 회원 : 게시물 작성자 시나리오

1. 앱 실행
2. 매칭 게시물 목록
    1. 카테고리 조회 (돌봄/친구찾기/물품나눔)
    2. 게시물 상태 카테고리 조회 (모집/매칭중/매칭완료)
3. 매칭 게시물에서 새글 작성 버튼
4. 게시물 작성
    1. 제목
    2. 카테고리(돌봄/친구찾기/물품나눔)
    3. 만나기로 한 날짜, 시간(24시간 기준) 설정
        1. 2026년 1월 28일 19시 30분
    4. 본문 작성
    5. 완료 버튼
5. 게시물 등록 완료
6. 게시물 선택
    1. 신청한 사람들 목록 보임
        1. 사람 클릭 시 팝업
            1. 사람 프로필
            2. 고양이 정보
            3. 리뷰 내용들
7. 매칭 선택 완료
    1. 신청 목록 중에서 1명 선택하면 매칭 완료
        1. 선택된 사람 상태: ‘매칭중’
        2. 선택되지 않은 상태: ‘실패’
    2. 매칭된 사람에게 게시물 올린 사람의 핸드폰 번호 공개
        1. 선택하면 선택한 사람의 ‘수락’ 상태 기다림
        2. 게시물 작성자는 신청자의 핸드폰 번호 모름
    3. 매칭된 사람이 ‘수락’ 확인 후
        1. 상태 변경 : ‘매칭 완료’
        2. 게시물 작성자는 ‘매칭 완료’ 상태면 매칭된 사람의 핸드폰 번호 확인 가능
8. 매칭 선택 변경
    1. 하루에 1번씩만 수정 가능
    2. ‘매칭 완료’ 상태이면 선택 변경 못함
9. 전화 번호로 연락 함
10. 게시물 기간 지난 후 신청자 리뷰 작성

### 회원 :  신청하려는 사람

1. 앱 실행
2. 매칭 게시물 목록
    1. 지역기반 조회(구로구/관악구/노원구)
    2. 카테고리 조회 (돌봄/친구찾기/물품나눔)
    3. 게시물 상태 카테고리 조회 (모집/매칭중/매칭완료)
3. 모집 상태 게시물 하나 선택
4. 신청자 내용 확인
5. 신청자 프로필 누름
    1. 사람 클릭 시 팝업
        1. 사람 프로필
        2. 고양이 정보
        3. 리뷰 내용들
6. 신청 버튼 누름
7. 신청 버튼 비활성, 신청 버튼 대신 상태 ‘대기중’으로 변경
8. 신청한 게시물 목록 조회
    1. 지역기반 조회(구로구/관악구/노원구)
    2. 카테고리 조회 (돌봄/친구찾기/물품나눔)
    3. 게시물 상태 카테고리 조회 (모집/매칭중/매칭완료)
9. 매칭 목록 조회
    1. 매칭중 && 본인이 선택을 받은 게시물 목록
    2. 수락 버튼
    3. 상대방의 핸드폰 확인 가능
10. 서로 연락
11. 게시물 일정 기간 지난 후 리뷰 작성
