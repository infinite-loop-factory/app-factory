# Country Tracker — Design Context

## Product
Personal travel log. Track visited countries, days stayed, visa-free limits.
Hero surface: world map with visited markers + share-as-image stats.

## Audience
- Primary: bilingual ko/en travelers (20s–40s), passport-conscious, CJK-literate
- Tech: smartphone-first, occasional desktop web for sharing
- Goal: low-friction logging, clean stat readout, shareable visual

## Platforms
- Mobile: Expo (iOS, Android) — touch primary
- Web: Expo Router web bundle — secondary, SSR enabled
- Min viewport: 375px (iPhone SE 3rd gen)
- Native chrome: gluestack-ui v3 + NativeWind v4

## Languages
- ko, en (CJK matters)
- Headings tested at 24/32px in Korean for line-break behavior

## Constraints
- No custom fonts shipped on native (perf budget) — use Pretendard Variable on web, system fallback on native
- Dark mode mandatory (DARK_MODE=media in dev/web)
- Existing primary palette is amber (#E69317 family) — reuse, do not retint
- Map is the visual hero — surrounding chrome must recede

## Reference Sites
- [linear.app](https://linear.app) — clean restraint, monochrome with single accent
- [stripe.com](https://stripe.com) — strong hierarchy, generous whitespace, purposeful animation

## Selected Direction
Stitch Hybrid — inherits the existing Stitch project (`13273652130168901163`,
"Country List Home Screen") baseline (amber accent, cream/paper surfaces,
Plus Jakarta Sans, card layout) and removes AI-slop residue (rainbow per-row
icons, equal-width stat hero, nested card decoration). See DESIGN.md §10 for
full divergence list.
