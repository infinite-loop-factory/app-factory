## 📝 기술 결정 사항

### 확정된 기술
- **프레임워크**: Expo SDK ~54
- **언어**: TypeScript 5.9+
- **스타일링**: NativeWind, Gluestack UI
- **라우팅**: Expo Router
- **상태 관리**: Context API
- **위치 서비스**: expo-location
- **캐싱**: AsyncStorage 또는 SQLite
